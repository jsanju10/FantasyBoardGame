Detailed Description

Represents the dynamic 2D game board on which players move.

The Board is responsible for:

  • Initializing a grid of BoardSquare objects.
  • Randomly populating squares with items and enemies.
  • Handling all operations related to a Player's interaction with the board: movement, pickup, drop, and attack.
  • Ensuring that movements occur only within bounds and that interactions correspond to the Player's current position.

Internally, the board is implemented as: std::vector<std::vector<std::unique_ptr<BoardSquare>>> which gives each square unique ownership and prevents accidental sharing.

PSEUDOCODE:

  1. Convert direction into (dx, dy).
  2. newX = player.x + dx newY = player.y + dy
  3. If NOT inBounds(newX, newY): return false.
  4. Update player's internal position.
  5. Return true.
Note

PSEUDOCODE:

  3. Call player.attack(enemy).
