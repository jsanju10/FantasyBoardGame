Detailed Description

Defines global constants and race stat templates used throughout the game.

This header provides:

  • The number of commands required to toggle between day and night.
  • A RaceStats struct describing the combat and survival attributes for each race.
  • Constant RaceStats presets for all player and enemy races, including day/night variations for Orcs.

These values centralize all core balance parameters, ensuring that character construction and race-specific behaviour remain consistent across the program.
