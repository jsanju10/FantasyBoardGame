Detailed Description

Concrete Item type that boosts health and strength.

Responsibilities:

  • Increase a Character's health and strength when picked up.
  • Remove those stat changes when dropped.
  • Store boost values for modification.

Rings satisfy assignment requirements: ✔ Unlimited carry quantity (subject to weight/strength rules) ✔ Stat-modifying behaviour via applyEffect/removeEffect ✔ Owned using std::unique_ptr<Item>
