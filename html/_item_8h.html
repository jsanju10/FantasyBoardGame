Detailed Description

Declares the Item abstract base class and ItemType enumeration for all in-game items.

Items represent all equippable or carriable objects in the game. They modify Character statistics through applyEffect() and removeEffect(). All concrete items (Weapon, Armour, Shield, Ring) derive from this class.

Items are ALWAYS passed, stored, and transferred via std::unique_ptr<Item>, ensuring exclusive ownership and preventing accidental sharing between characters or board squares.

The Item class also provides the static factory createRandomItem(), used when the board is populated randomly during initialization.
